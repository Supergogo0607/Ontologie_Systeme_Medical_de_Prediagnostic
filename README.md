# Introduction de Projet
Une Ontologie pour un Système Médical de Prédiagnostic.<br>
Problèmes visés :<br>
1) Identiffer les affections potentielles dont ils pourraient souffrir.<br>
2) Les orienter vers le service médical le plus adéquat pour une consultation.<br>
3) Suggérer les médicaments usuels pouvant être indiqués pour leur pathologie présumée.<br>
